Decorative images are now discovered automatically from `assets/` by file name.

Use this naming pattern:
- `image-1.png`
- `image-2.png`
- `image-3.png`

Any `image-*.png` found under `assets/` (including subfolders) can be selected randomly in the app.

## Credits
- These decorative PNG assets are bundled local files used by the sidebar artwork system.
- Their original external source URLs and license details are not currently documented in this repository.
- Before redistributing the app broadly, either:
  - add source URL and license information for each `image-*.png`, or
  - replace the files with artwork whose provenance is documented.
- See the main [README](../../README.md) credits section for the current media-policy note.
